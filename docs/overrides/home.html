Tux

Tux is an all-in-one open source Discord bot, originally designed for the All Things Linux community. It is designed to provide a variety of features to the server, including moderation, support, utility, and various fun commands.

Why Tux?

Built for Linux Communities

Tux combines powerful moderation tools with engaging features, all built with modern Python and best practices.

  • {% include ".icons/material/shield-half-full.svg" %} Moderation

    Comprehensive tools with case tracking, jail system, and audit logs

  • {% include ".icons/material/lightning-bolt.svg" %} Performance

    Async architecture optimized for large servers with hot-reload

  • {% include ".icons/material/gamepad-variant.svg" %} Engagement

    XP system, starboard, bookmarks, and customizable role rewards

  • {% include ".icons/material/cog.svg" %} Configuration

    Flexible setup with TOML/YAML/JSON and interactive wizard

  • {% include ".icons/material/chart-line.svg" %} Analytics

    Sentry integration, InfluxDB metrics, and detailed logging

  • {% include ".icons/material/puzzle.svg" %} Extensible

    Plugin system for custom features and integrations
