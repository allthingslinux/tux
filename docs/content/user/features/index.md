# Features

Tux provides automated features that enhance your Discord server experience. These features work in the background to improve engagement, moderation, and user experience.

## Available Features

### Community & Engagement

- **[XP & Leveling](leveling.md)** - Reward active members with XP and automatic role progression
- **[Starboard](starboard.md)** - Highlight popular messages in a dedicated channel
- **[Status Roles](status-roles.md)** - Assign roles based on Discord custom status messages

### Utility & Organization

- **[Temp VC](temp-vc.md)** - Create temporary voice channels on demand
- **[Bookmarks](bookmarks.md)** - Save important messages to your DMs

### Moderation

- **[GIF Limiter](gif-limiter.md)** - Prevent GIF spam with automatic rate limiting

## Getting Started

Most features require configuration in your server's `config.toml` file. See the [Admin Guide](../../admin/config/index.md) for setup instructions.
