## Core Features

| Moderation Tools | Engagement Features |
|------------------|-------------------|
| **Case Management** - Track all moderation actions | **XP System** - Activity-based leveling |
| **Jail System** - Temporary restriction system | **Leaderboards** - Server rankings |
| **Auto-moderation** - Automated content filtering | **Role Rewards** - Automatic role assignment |
| **Audit Logs** - Complete action history | **Starboard** - Highlight best content |
| **Warning System** - Progressive discipline | **Custom Commands** - Server-specific commands |

| Configuration | Analytics & Monitoring |
|---------------|----------------------|
| **Multi-format Support** - TOML, YAML, JSON | **Sentry Integration** - Error tracking |
| **Setup Wizard** - Interactive configuration | **InfluxDB Metrics** - Performance monitoring |
| **Hot Reload** - Update without restart | **Detailed Logging** - Loguru-powered logs |
| **Environment Variables** - Docker-friendly | **Command Statistics** - Usage analytics |
| **Per-guild Settings** - Customize per server | **Database Insights** - Query performance |
