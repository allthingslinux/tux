# Frequently Asked Questions

Common questions about Tux.

## General

### What is Tux?

Tux is a Discord bot for the All Things Linux community, featuring moderation, XP/leveling, snippets, and more.

### Is Tux open source?

Yes! Licensed under GPL-3.0. See [GitHub](https://github.com/allthingslinux/tux).

### Can I self-host Tux?

Yes! See [Deployment Guide](../admin-guide/deployment/).

### Which Python version?

Python 3.12+ required.

## Features

### What commands are available?

See [Commands Documentation](../user-guide/commands/moderation.md).

### How does the XP system work?

See [XP System Guide](../user-guide/features/xp-system.md).

### What's the permission system?

Tux uses ranks 0-7. See [Permissions Guide](../user-guide/permissions.md).

## Technical

### What database does Tux use?

PostgreSQL 12+.

### What libraries?

- discord.py
- SQLModel/SQLAlchemy
- Pydantic
- Loguru
- Sentry (optional)
